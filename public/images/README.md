# คำแนะนำการเพิ่มรูปภาพและโลโก้

## โครงสร้างโฟลเดอร์
โฟลเดอร์นี้สำหรับเก็บรูปภาพและโลโก้ต่างๆ ที่ใช้ในหน้าแรก

## รายการไฟล์ที่ต้องเพิ่ม

### โลโก้ AI Tools (สำหรับ AI Tools Carousel)
กรุณาเพิ่มโลโก้จริงของ AI แต่ละตัว ขนาดแนะนำ: 200x200px หรือ 512x512px (PNG โปร่งใส)

- `chatgpt.png` - โลโก้ ChatGPT
- `claude.png` - โลโก้ Claude AI
- `gemini.png` - โลโก้ Google Gemini
- `midjourney.png` - โลโก้ Midjourney
- `suno.png` - โลโก้ Suno AI
- `veo.png` - โลโก้ VEO 3.1
- `nano-banana.png` - โลโก้ Nano Banana
- `infinite-talk.png` - โลโก้ Infinite Talk

### รูปโค้ช (สำหรับ Coaches Section)
กรุณาเพิ่มรูปถ่ายของโค้ช ขนาดแนะนำ: 800x800px หรือสัดส่วน 1:1

- `coach-film.jpg` - รูปถ่าย Coach FILM (ผู้เชี่ยวชาญด้านการทำคลิป AI)

### ไฟล์ Lottie Animations (JSON) - Optional
หากต้องการใช้ Lottie animations แทน emoji:
- `coach.json` - ไอคอนโค้ช
- `ai-tools.json` - ไอคอน AI tools
- อื่นๆ ตามต้องการ

## หมายเหตุ
- ไฟล์รูปภาพควรมีคุณภาพสูงแต่ไฟล์ไม่ใหญ่เกินไป (< 500KB ต่อรูป)
- ใช้ชื่อไฟล์เป็นตัวพิมพ์เล็กและใช้ - แทนช่องว่าง
- หากใช้โลโก้ของบริษัทอื่น กรุณาตรวจสอบลิขสิทธิ์ก่อนใช้งาน

## ข้อมูลโค้ช
ปัจจุบันมีโค้ช 1 ท่าน:
- **Coach FILM** - ผู้เชี่ยวชาญด้านการทำคลิป AI
  - สอน ChatGPT, Veo3.1, ElevenLabs, Comfy UI, n8n, CapCut และอื่นๆอีกมากมาย
